# Curso sobre uso de versionamento com GIT.
 Aprendendo e praticando versionamento com o git e github, utilizando diversos comandos como:
 
 - git init, clone
 - git add . git add --all
 - git status
 - git checkout git checkout -b 'nome'
 - git branch
 - git commit -m git commit -am
 - git log
 - git ignore
 - git diff
 - git push
 - git pull
 - git fetch
 - git restore
 - git rm
